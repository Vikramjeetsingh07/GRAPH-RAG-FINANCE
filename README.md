# GRAPH-RAG-FINANCE
Knowledge Graph Enhanced Retrieval-Augmented Generation for Finance Data.

## What This Project Demonstrates

- **Knowledge Graph**: 25+ finance entities with typed relationships
- **Graph RAG Pipeline**: Entity extraction → Graph traversal → Chunk retrieval → Answer generation
- **Force-Directed Visualization**: Interactive graph with physics simulation
- **Finance Domain**: Companies, sectors, concepts, events, and metrics
